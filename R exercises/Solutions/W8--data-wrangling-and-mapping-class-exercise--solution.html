Exploring the ESS (European Social Survey) further

The ESS is a huge survey run every two years across Europe, focusing on different social and political attitudes, as well as personal characteristics. Here we use Wave 7 from 2014, which collected 601 variables on 40,185 respondents and included a special module on the social determinants of health.

Today we will focus on replicating and extending some analyses from Thomson et al.(2017) who considered regional inequalities. At this stage, we will ignore two important aspects to focus on fundamental data manipulation skills: - we will ignore survey weights, which need to be used to make country results representative of the countries population, - we will ignore different types of missing data. I have recoded all missing values as NA; in a more thorough analysis, the different reasons for missingness reported in the codebook should be maintained.

Initial data preparation

  • Filter to include only participants who are at least 18 years old
  • Create a bmi variable: weight(in kg)/height(in m)^2 (^2 means squared, make sure to convert height to meters)
  • Remove Israel (IL) as they do not provide information on subregions (and are not on the European map)
  • Create a poor_health variable that is TRUE if someones health is 4 (bad) or 5 (very bad) and FALSE otherwise
#Note that I am also adding a country_name variable - that makes plots easier to read than if we just use the codes includes in cntry

ess <- ess %>%
  filter(agea >= 18, cntry != "IL") %>%
  mutate(bmi = weight / (height / 100)^2, poor_health = health > 3,
         country_name = as_factor(cntry))

Create a scale for depression

Depression was measured with 8 items. We need to calculate their mean, but should also check whether the scale is internally consistent.

To test a scale for internal consistency, you need to calculate Cronbachs alpha. To do that easily, you will need to install a package that has a specific function for it; we will use the psych package. So run install.packages(psych) in the console.

Note: Given that we will only use a single function from that package, I would recommend typing psych::alpha() rather than loading the full package, but you could as well run library(psych) and then use the alpha() function.

The proposed depression scale consists of the following items:

fltdpr flteeff slprl wrhpp fltlnl enjlf fltsd cldgng

# Two items should be reverse coded because they do not measure high but low levels of depression. Have a look at the variable key pdf to identify which, and then create reverse-coded versions by subtracting them from 1 + the maximum. var1 below needs to be replaced by one of the variable names, and then you need to create the same command for the other

ess <- ess %>% mutate(enjlf_rev = 5 - enjlf, wrhpp_rev = 5 - wrhpp)

#Test whether the 8 items listed above can form an internally consistent scale
#Here, I am extracting the relevant field from the psych::alpha() output. That saves me from printing lots of irrelevant output. To figure out what to extract, use the str() function on the output - but that part is not essential
ess %>% select(fltdpr, flteeff, slprl, wrhpp_rev, fltlnl, enjlf_rev, fltsd, cldgng) %>% psych::alpha() %>% magrittr::extract2("total") %>% magrittr::extract("raw_alpha")
##  raw_alpha
##  0.8281315
#Create a new depression variable that is the sum of the eight. Ensure that you include the reverse-coded ones. Avoid using the sum() function inside mutate() - just add them up.
#Here I am using this rather complicated code to be able to include cases where some depression questions haven't been answered (na.rm = TRUE). You can also use the + sign, but it will give you a slightly smaller sample.

ess <- ess %>% mutate(depression_score = rowSums(select(., fltdpr,  flteeff,  slprl , wrhpp_rev , fltlnl , enjlf_rev , fltsd , cldgng), na.rm = TRUE))

#Calculate and plot the average depression score by country 
#Remember that you can use group_by() before summarise() to create summaries per group
#A column chart `geom_col()` might be best; you can add `+ coord_flip()` to get horizontal bars with better-to-read labels
#If you want to arrange the bars, use the reorder function on the x-axis, e.g., ggplot(aes(x=reorder(cntry, depression_score), y=depression_score))

#Note that I do not print but save the plot - then I can show them side-by-side in a moment.

Map health and depression per region

In the paper, they used deciles to rank regions. We will use relative odds for being in poor health or depressed. But lets first consider how to draw maps.

Drawing maps

For ggplot, maps are polygons that consist of individual lines - drawn with geom_polygon(). To account for the fact that degrees of longitude and latitude cover very different distances, especially up north, we need to use a map projection, offered by coord_map(). We also need to zoom in on the European mainland, ignoring overseas territories and the like. For coord_map() we need to choose a good map projection - you can have a look here to see some of them in action. The European Environmental Agency suggests to use an azimuthal equal-area projection for statistical analysis as it more adequately represents the size of places, so lets go for that here.

Typically, only fill is used to convey information on maps. So you can copy-paste the code below and replace fill and labels to create meaningful maps.

map_mercator <- geo_data %>% 
  ggplot(aes(x = long, y = lat, group = group, fill = 1)) +
  geom_polygon(color = "grey", size = .25) +
  coord_map(xlim = c(-11, 30), ylim = c(35, 72), projection = "mercator") + labs(title = "Mercator projection")

map_equal_area <- geo_data %>% 
  ggplot(aes(x = long, y = lat, group = group, fill = 1)) +
  geom_polygon(color = "grey", size = .25) +
  coord_map(xlim = c(-11, 30), ylim = c(35, 72), projection = "azequalarea") + labs(title = "Equal-area projection")

map_mercator + map_equal_area

Calculate relative risk

Calculate the average share of people who are depressed or in poor health across the dataset. Then calculate the share per region health_region and divide that by the overall share to get a relative risk. Finally, plot two maps with regions filled by these relative risks.

Please note that I am adding quite a lot of formatting to the maps here - not all of that is essential, but I wanted to show you how to get close to a publishable product. If you do not understand what some element of the ggplot-code is doing, a good way to find out is to delete it and see what changes.

#Here we should really use weights once - pweight adjusts for unequal population sizes between countries. Without that, Europe-wide averages make very little sense.
total_depressed <- weighted.mean(ess$depressed, ess$pweight, na.rm = TRUE)
total_poor_health <- weighted.mean(ess$poor_health, ess$pweight, na.rm = TRUE)

# create correct labels for the axes
geo_labels <- tibble(
  eastwest_breaks = seq(-10, 30, 10),
  northsouth_breaks = c(seq(40, 70, 10), NA)
)

geo_labels <- geo_labels %>% mutate(
  eastwest_labels = case_when(
    eastwest_breaks < 0 ~ paste0(abs(eastwest_breaks), " W"),
    eastwest_breaks > 0 ~ paste0(eastwest_breaks, " E"),
    TRUE ~ paste0(eastwest_breaks, "")
  ),
  northsouth_labels = paste0(northsouth_breaks, " N")
)

# Set limits on risk range (otherwise, extremes will dominate colours)
risk_range <- c(0.5, 2)

risk_labels <- tibble(
  breaks = seq(risk_range[1], risk_range[2], 0.5),
  labels = breaks
)

# Label extremes as smaller/greater than
risk_labels$labels[1] <- paste("<=", risk_labels$labels[1])
risk_labels$labels[length(risk_labels$labels)] <- paste(">=", risk_labels$labels[length(risk_labels$labels)])

#Helper function to allow 1 (i.e. being on the European average) to be the midpoint of the fill scale
mid_rescaler <- function(mid = 1) {
  function(x, to = c(0, 1), from = range(x, na.rm = TRUE)) {
    scales::rescale_mid(x, to, from, mid)
  }
}


map_depressed <- ess %>%
  #Calculate relative risk per region
  group_by(health_region) %>%
  summarise(odds_depressed = mean(depressed, na.rm = TRUE) / total_depressed, N = n()) %>%
  #Limit it to established risk range
  mutate(odds_depressed = ifelse(odds_depressed > risk_range[1], odds_depressed, risk_range[1]), odds_depressed = ifelse(odds_depressed < risk_range[2], odds_depressed, risk_range[2])) %>%
  #Filter for regions with minimum number of observations
  filter(N >= 50) %>%
  #Join to geo_data (coordinates)
  right_join(geo_data) %>%
  ggplot(aes(x = long, y = lat, group = group, fill = odds_depressed)) +
  geom_polygon(color = "grey", size = .25) +
  #Define map-projection
  coord_map(xlim = c(-11, 30), ylim = c(35, 72), projection = "azequalarea") +
  #Define fill colours and scale labels
scale_fill_distiller(palette = "RdYlGn", na.value = "white", limits = c(risk_range[1] - 0.1, risk_range[2] + 0.1), breaks = risk_labels$breaks, labels = risk_labels$labels, rescaler = mid_rescaler()) +
  #Define labels for x- and y-axes
  scale_x_continuous(breaks = geo_labels$eastwest_breaks, labels = geo_labels$eastwest_labels) +
  scale_y_continuous(breaks = geo_labels$northsouth_breaks, labels = geo_labels$northsouth_labels) +
#Note that \n adds a line break into the title
    labs(x = "", y = "", fill = "Relative risk", title = "Inequality in \nrate of depression")
## `summarise()` ungrouping output (override with `.groups` argument)
## Joining, by = "health_region"
map_health <- ess %>%
  group_by(health_region) %>%
  summarise(odds_poor_health = mean(poor_health, na.rm = TRUE) / total_depressed, N = n()) %>%

  filter(N >= 50) %>%
  right_join(geo_data) %>%
  ggplot(aes(x = long, y = lat, group = group, fill = odds_poor_health)) +
  geom_polygon(color = "grey", size = .25) +
  coord_map(xlim = c(-11, 30), ylim = c(35, 72), projection = "azequalarea") +
  scale_fill_distiller(palette = "RdYlGn", na.value = "white", limits = c(risk_range[1] - 0.1, risk_range[2] + 0.1), breaks = risk_labels$breaks, labels = risk_labels$labels, rescaler = mid_rescaler()) +
  scale_x_continuous(breaks = geo_labels$eastwest_breaks, labels = geo_labels$eastwest_labels) +
  scale_y_continuous(breaks = geo_labels$northsouth_breaks, labels = geo_labels$northsouth_labels) +
  labs(x = "", y = "", fill = "Relative risk", title = "Inequality in \nrate of poor health")

map_health + map_depressed + plot_layout(guides = "collect")
